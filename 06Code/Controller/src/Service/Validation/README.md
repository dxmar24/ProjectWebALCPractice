# Validation Services

Validators keep request rules out of controllers and models.

Each validator focuses on one request concept, returns field-level errors, and avoids database writes.

Current validation areas:

- Enrollment requests.
- Student profiles.
- Ecuadorian national ID check digits.
- Student profile photos.
- Teacher accounts.
- Attendance records.
- Class plans.
- Finance reports.
- Professional events.
- Dancer event assignments.
